# copy-path.yazi

一个 Yazi 插件，可以复制文件/文件夹的路径信息到剪贴板。

## 功能

### 单文件（无选中或仅聚焦一项）

对于文件：
- 复制完整路径
- 复制文件名
- 复制基础名（不含扩展名）
- 复制扩展名
- 复制相对路径（相对于 Yazi 启动路径）

对于文件夹：
- 复制完整路径
- 复制文件夹名
- 复制相对路径（相对于 Yazi 启动路径）

软链接会额外追加「Symlink target」选项。

### 多文件（选中多项）

仅保留对多文件有意义的选项，以换行分隔合并后复制：
- 复制完整路径
- 复制文件名
- 复制相对路径

路径按字母序排序。

## 安装

终端中执行：

```bash
ya pkg add gogongxt/copy-path
```

## 用法

在您的 `keymap.toml` 中添加以下按键绑定：

```plaintext
[mgr.prepend_keymap]
{ on = "y", run = "plugin copy-path", desc = "Copy file path info" },
```

使用方法：
1. 选中文件或文件夹
2. 按下绑定的按键（如 `y`）
3. 选择要复制的路径类型（数字键 1-5）
4. 内容将自动复制到剪贴板
